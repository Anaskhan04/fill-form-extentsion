# AutoForm Pro üöÄ

**AutoForm Pro** is a powerful, privacy-focused browser extension that automates filling out web forms. Unlike standard autofillers, it uses intelligent fuzzy matching to detect fields even when they don't have standard names, and it supports custom user profiles for different contexts (e.g., Personal, Work, Shopping).

## ‚ú® Features

- **Smart Field Detection**: Uses fuzzy logic to identify fields like "First Name", "ZIP Code", "Job Title", etc., even if the website uses non-standard labels.
- **Context-Aware**: Distinguishes between similar fields (e.g., "Address Line 1" vs "Address Line 2").
- **Radio & Checkbox Support**: Intelligently handles "Yes/No" questions, Gender selections, and Terms & Conditions.
- **Multiple Profiles**: Create and switch between unlimited profiles (e.g., "Anas - Student", "Anas - Developer").
- **Privacy First**: All data is stored locally on your device. Nothing is sent to the cloud.
- **Instant Control**:
  - Right-click context menu: **"AutoFill Now"**
  - Keyboard Shortcut: `Ctrl+Shift+L` (Windows/Linux) or `Cmd+Shift+L` (Mac)
  - **Clear Fields** button to instantly reset a form.

## üì• Installation

Since this extension is in developer mode, follow these steps to install it:

1.  **Download/Clone** this repository to your computer.
2.  Open **Google Chrome** (or Edge/Brave).
3.  Navigate to `chrome://extensions/`.
4.  Toggle **Developer mode** in the top right corner.
5.  Click **Load unpacked**.
6.  Select the folder where you saved this project.
7.  Done! The AutoForm Pro icon should appear in your toolbar.

## üõ†Ô∏è Usage

1.  **Setup**: Click the extension icon and configure your "Default" profile with your details (Name, Address, Job, etc.).
2.  **Fill**: Navigate to any form on the web.
    - Click the extension icon and hit **Fill Now**.
    - OR Rights-click and select **AutoFill Now**.
    - OR Press `Ctrl+Shift+L`.
3.  **Manage**: Use the popup to add new profiles or clear fields if a mistake is made.
